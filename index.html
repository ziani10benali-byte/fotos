Explicación Técnica

¿Por qué no se puede apagar un dispositivo desde una web?

  • Sandboxing: Los navegadores ejecutan código en un entorno aislado
  • Política de mismo origen: Restringe acceso entre sitios
  • Permisos de sistema: JavaScript no tiene acceso a comandos de apagado
  • Protecciones del SO: iOS/Android bloquean acciones peligrosas

Informe Técnico

Metodología

Este proyecto simula vectores de ataque que fueron reales en el pasado pero que han sido parcheados. Se utiliza HTML, CSS y JavaScript para crear demostraciones educativas que muestran:

  1. Cómo los atacantes explotaban vulnerabilidades
  2. Qué protecciones existen hoy en día
  3. Por qué es importante mantener los dispositivos actualizados

Hallazgos principales

  • Los navegadores modernos tienen múltiples capas de seguridad
  • El sandboxing previene el 99% de los ataques web
  • Las actualizaciones de seguridad son críticas
  • La educación del usuario es la mejor defensa

Código ético

Todo el código en este proyecto:

  • Es 100% educativo y de demostración
  • No ejecuta comandos reales del sistema
  • No causa daño a dispositivos
  • Se ejecuta solo en el navegador
  • Cumple con las leyes de ciberseguridad
